## ✨ Features

- **📊 Dashboard Overview**: Displays elegant summary cards (Total Balance, Target Income, Total Expenses).
- **📉 Interactive Visualizations**: Powerful `recharts` based visualizations for Balance Trends (Line Chart) and Expense Categories (Pie Chart).
- **📝 Transactions Management**: Complete list of transactions with capabilities to Sort, Filter (by type/search/date), and Add/Edit/Delete records.
- **🛡️ Role-based Access**: Easily switch between **Viewer Mode** (Read-only) and **Admin Mode** (Create, Edit, Delete transactions) via the top header segmented control.
- **💡 Insights Generation**: Automatically generates simple insights based on your transaction history (e.g., your highest spending category).
- **🌙 Dark Mode**: Integrated light and dark themes using deeply beautiful vanilla CSS variables.
- **💾 State Persistence**: Transactions and theme settings are safely persisted in your browser's Local Storage.
- **📥 CSV Exporting**: Easily download a CSV of your filtered scoped transactions with one click!

## 🚀 Setup Instructions

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Start the development server:**
   ```bash
   npm run dev
   ```

3. **Enjoy your dashboard locally at** `http://localhost:5173/`

## 🛠️ Tech Stack
- **ReactJS**
- **Vite**
- **Recharts** (Charting)
- **Lucide React** (Icons)
- **Vanilla CSS** (Theming / Layout, no external frameworks necessary!)
